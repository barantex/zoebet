# ZoeBet — Product Overview

ZoeBet is a Turkish-language online betting and casino platform. It provides:

- **Sports betting** — live and pre-match odds across football, basketball, tennis, e-sports, etc.
- **Casino & Live Casino** — game catalog from multiple providers
- **Promotions & Tournaments** — marketing content managed via admin back office
- **User accounts** — registration, login, wallet balance, transaction history
- **Admin back office** — manage matches, games, promotions, and banners

The UI language is Turkish. All user-facing strings, error messages, and labels should be written in Turkish.

The admin account is identified by the email `admin@zoebet.com`.
